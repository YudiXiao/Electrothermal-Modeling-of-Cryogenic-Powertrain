# Electrothermal-Modeling-of-Cryogenic-Powertrain
This repository contains code and data for modeling power loss and coolant temperature of a conceptualized hydrogen-powered narrow-body all-electric aircraft flying between Scotland and Spain.

The code needs to preload LTspice simulation results to perform modeling the inverter part of the powertrain. An example preload-data can be accessed from here:https://drive.google.com/file/d/1dWNONoa_bT9NsxCUxqX96-AgPtj99XsE/view?usp=sharing
